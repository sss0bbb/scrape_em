# scrape_em

scrape data from a hard to navigate site

## Dependencies

1. python2.7
2. requests `pip install requests` 
3. beautifulsoup `pip install bs4`

## Usage

`python scrape_em.py -u <url>`
`python scrape_em.py -h` for full options
